# Лабораторная работа №8 (Spring Boot + MySQL + Docker)

### Как запустить:
1. Убедитесь, что установлен Docker Desktop.
2. Выполните команду в корневой папке:
   ```bash
   docker-compose up --build
   ```
3. Приложение доступно по адресу: http://localhost:8080


4. Если не запускается может помочь следующая конфигурация Docker Engine:
```
{
  "registry-mirrors": [
    "https://timeweb.cloud"
  ]
}
```
